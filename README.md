# Multi-Agent AI Assistant

An intelligent multi-agent system built using Google ADK, Gemini 2.5 Flash, ChromaDB, and Sentence Transformers. The assistant routes user queries to specialized agents for reasoning, calculations, and retrieval-augmented question answering.

## Features

* Intent-based query routing
* Multi-agent architecture with specialized agents
* Retrieval-Augmented Generation (RAG)
* Semantic search using vector embeddings
* Tool calling for calculations and task execution
* Context-aware responses using ChromaDB
* Interactive Streamlit interface

## Architecture

User Query → Root Agent → Specialized Agent → Response

Specialized Agents:

* Greeting Agent
* Math Reasoning Agent
* RAG Knowledge Agent

## Tech Stack

* Python
* Google ADK
* Gemini 2.5 Flash
* ChromaDB
* Sentence Transformers
* Streamlit

## Results

* Evaluated on 30+ test queries
* Achieved approximately 80% intent-routing accuracy
* Successfully handled reasoning, retrieval, and conversational tasks through a unified interface
